#Markdown Editor
## Manage your markdown bins and share it with other users
### Prerequisits
- node.js installed

###Install
- clone this project
 cd to merkbin
```sh
npm install
meteor
```

visit http://localhost:3000
- you can switch to prodmode and choose another port

###Usage
- Create an account
- login and test the markbin editor
- Share your bins with other users
